本说明对应当前主流程（run_full_selection_pipeline.py + results_io.py）拆分后的 JSON 字段。
每个数据集输出目录结构示例：outputs/<批次>/<数据集名>/

================================================================================
1) summary.json（主对比表，论文/汇总脚本直接读取）
================================================================================
顶层字段：
- 任务说明：流程概述（外层10折，同折 train 约简 + test 评估，按参数选最优后汇总）。
- 数据集：CSV 绝对路径。
- 目标列：分类标签列名。
- 样本数 / 特征数 / 折数：整型（折数固定为 10）。

- 算法最优参数效果（10折，多模型）：字典，键为算法名；每个算法一条记录，字段如下：

  - 最优参数：
      在该算法参数网格中，按 score(p) 选出的参数组合。
      score(p) = 先对每个分类器在 10 折上求指标均值，再对所有分类器求均值（主看 acc）。

  - 平均指标：
      最优参数 p* 下的多模型聚合指标（acc、f1_macro、precision_macro、recall_macro 等及 *_std）。
      不含 folds.json 里 LR 单模型那条路径；summary 的 acc 来自多模型评估。

  - 候选子集总数：
      最优参数 p* 在 10 折中产生的候选条数（每折通常 1 条，故通常 ≈ 10）。
      若某折该参数未产出有效子集（空子集被跳过），则小于 10。

  - 平均约简子集长度：
      仅针对最优参数 p*：10 折各候选子集长度的算术平均（len_sum / count）。

  - 算法参数网格搜索总耗时秒：
      该算法在本数据集上、跨 10 折、跑完「全部参数组合」的约简阶段总耗时（秒）。
      不含分类器训练/预测，不含读盘与写 JSON。
      注意：这是「全网格搜索成本」，不是「仅最优参数 p*」的约简时间。
      ARPDMF 另不含：Gaussian PDMF 模糊化、模糊数批归一化、样本熵矩阵 E_mat 构建；
      上述预处理按 (k,μ) 每折最多各算 1 次（与 δ 无关），不计入本字段。
      本字段对 ARPDMF 仅累计三阶段属性约简（Sig_out / Sig_in / δ）时间。
      与 sub_info/times.json 中「约简阶段_按算法汇总[算法名]」数值一致。

  - 覆盖折数：
      最优参数 p* 在多少折上有有效候选记录（通常 10）。

  - rank：
      竞赛排名（1,2,2,2,5）：仅当 平均指标.acc 完全相同时名次相同，下一名次不跳空。
      summary 中算法键的排序顺序为：acc 降序 → 平均约简子集长度升序 →
      算法参数网格搜索总耗时秒升序 → 最优参数字典序（后三项用于并列时的展示顺序，不改变同 acc 的 rank）。

未启用或未跑出候选的算法：候选子集总数=0，其余指标多为 0，汇总 Markdown 脚本会跳过该列。

================================================================================
2) full_pipeline_result.json（完整大 JSON，含上述 summary 源数据）
================================================================================
除 summary 相关外，常用块：
- 参数网格：各算法超参候选。
- 每折记录：同 sub_info/folds.json。
- 按算法结果_最优参数：写入 summary 的原始对象（写入 summary 前会加 rank）。
- 模型_按算法：各算法 × 各分类器的辅助统计（全候选摊平，非 summary 主口径）。
- ARPDMF_模型指标序列：供 arpdmf_by_params.json 多模型聚合。
- 耗时秒：约简阶段 / 约简阶段_按算法汇总 / 评估阶段 / 总耗时。

已取消输出：按算法结果_折内均值、algorithms_overview.json、algo_*.json。

================================================================================
3) arpdmf_by_params.json（与 summary.json 同目录，仅 ARPDMF）
================================================================================
- 说明：按 (delta, k, mu_method) 分组；平均指标与 summary 中 ARPDMF 多模型口径一致。
- 按参数组合：数组，每项含
  - 参数、有效折记录数、平均指标、平均约简子集长度
  - 平均约简耗时秒、约简耗时秒_合计（该参数组合在 10 折上的约简耗时，非全算法网格总时间）

================================================================================
4) models_overview_by_algorithm.json（数据集根目录，辅助）
================================================================================
每个算法下 per_model / per_group：对「全部候选、不去重」摊平后的均值，用于细查，非 summary 主表。

================================================================================
5) sub_info/param_grids.json
================================================================================
各属性约简算法的参数网格（与 config.get_param_grids 一致）。

================================================================================
6) sub_info/folds.json
================================================================================
每折记录数组，元素含：
- 折号：1..10
- 约简耗时秒：字典，算法名 → 该折该算法「整段约简」耗时（该折内全参数网格合计，秒）
- 候选数量：该折明细条数
- 明细：数组，每条含
  - 折号、算法、参数、子集（特征名列表）
  - 指标：该折 test 上 LogisticRegression 的 acc/f1_macro 等（LR 快速评估）
  - 准确率：同 指标.acc
  - 约简耗时秒（可选）：该条候选对应的一次约简耗时；FNRS 为折内总耗时按候选数均分近似

================================================================================
7) sub_info/unique_subsets.json
================================================================================
- 去重后子集总数 / 去重后子集：
  按子集内容去重后的统计（出现次数、来源、折准确率、平均准确率等）。
  与 summary 的「按算法最优参数」口径不同，仅供子集层面分析。

================================================================================
8) sub_info/times.json
================================================================================
- 约简阶段：所有启用算法在 10 折上的约简总耗时之和（秒）。
- 约简阶段_按算法汇总：每个算法 10 折约简总耗时；与 summary「算法参数网格搜索总耗时秒」一致。
- 评估阶段：10 折分类评估（含多模型循环）总耗时（秒）。
- 总耗时：从读数据到写 JSON 的 wall-clock（秒）。

================================================================================
9) sub_info/mfnmi_param_surface.json（仅启用 MFNMI 时）
================================================================================
MFNMI 参数面：delta、theta、acc_mean、count。

================================================================================
指标口径速查（避免混淆）
================================================================================
| 量                     | 范围           | 是否仅最优参数 |
|------------------------|----------------|----------------|
| summary.acc            | 最优参数 p*    | 是             |
| 平均约简子集长度       | 最优参数 p*    | 是             |
| 算法参数网格搜索总耗时秒 | 全参数×10折  | 否（全网格；ARPDMF 另不含模糊化预处理） |
| folds 明细.准确率      | 每条候选       | 否（LR 单模型）|

重跑实验后 summary 才为新字段；历史 outputs 中旧 summary 可能仍含已废弃字段名。
